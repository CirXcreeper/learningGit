切换分支(创建新分支)：
        切换： git checkout <your_branch>       创建：  git checkout -b <new_branch>
或:     git branch <your_branch>
或:     git switch <your_branch>    // recommended

查看分支：
        git branch

合并目标分支至当前分支：
        git merge <target_branch>


删除不需要的分支：
        git branch -d <branch_to_delete>
